# Security policy

Atlas is a portfolio/reference project and accepts reports for the current `main` branch. It does not operate real customer accounts, money, identity documents, or cardholder data.

Please report suspected vulnerabilities privately through [GitHub private vulnerability reporting](https://github.com/MichaelSeveen/atlas/security/advisories/new). Do not open a public issue before coordination, and do not include real personal data, credentials, payment data, or destructive proof-of-concept activity. Use synthetic identifiers and the minimum reproduction needed.

Reports are triaged under `docs/runbooks/VULNERABILITY_DISCLOSURE.md`. Acknowledgement, remediation, disclosure, credit, or bounty timing is not guaranteed; no bounty program or production-security certification is implied.
